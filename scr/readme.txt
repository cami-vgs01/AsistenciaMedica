Esta carpeta contiene todo el código fuente del proyecto dividido por módulos 
(solo por fines de ejemplificación se considera una arquitectura MVC, 
por lo que esta no rige la arquitectura final que se decida tener para el mismo).

Fuera de la misma existirá una carpeta "bin" correspondiente a los binarios de 
ejecución de las distintas versiones de la aplicación del proyecto.
